# Rakeda.com

> Formidable.


## Installation & Build

You need [node.js](http://nodejs.org/) installed on your system.

Install project requirements:

```sh
npm -g install grunt-cli  # needs admin rights
npm install
```

Build project:

```sh
grunt
```

Start development server:

```sh
./server
```
